# 🌤️ Weather Information Web Application

A simple, clean, and responsive weather web application that allows users to fetch real-time weather data for any city around the world. Built using HTML, CSS, and Vanilla JavaScript.

## ✨ Features

- **Real-time Weather Data:** Enter any city name to get the current weather.
- **Dynamic Display:** Shows Temperature (°C), Weather Condition, and Humidity level.
- **Error Handling:** Displays a user-friendly error message if an invalid city name is entered.
- **Loading State:** Shows a loading indicator while fetching data from the API.
- **Clean UI:** Simple and intuitive user interface.

## 🛠️ Technologies Used

- **HTML5:** For the structure of the web page.
- **CSS3:** For styling and layout.
- **JavaScript (ES6):** For DOM manipulation and asynchronous API fetching (`fetch`).
- **OpenWeatherMap API:** To retrieve real-time weather data.

## 🚀 How to Run Locally

Follow these steps to run this project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/sunilGupta71](https://github.com/sunilGupta71/Skillifyz-Task---3)

2.  Get your Free API Key:

Go to OpenWeatherMap and create a free account.

Generate your API key from the dashboard.

3.  Add your API Key:

Open the script.js file in your code editor.

Replace the placeholder 'YOUR_API_KEY_HERE' with your actual OpenWeatherMap API key.

4.  Run the app:

You can simply double-click the index.html file to open it in your browser.

Alternatively, for the best experience, open the project in VS Code and use the Live Server extension.
